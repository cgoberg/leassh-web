Screen Time

Know what they're doing

Power Users

Fleet monitoring for homelabbers and self-hosters

SSH-based monitoring with rules, webhooks, and actuation. No agents required.

Fleet Monitoring

SSH into every machine

CPU, GPU, RAM, disk, processes, idle time, temperatures — probed automatically every two minutes. Windows, macOS, Linux. No agent required.

Rules & Triggers

Define rules in YAML

When GPU load drops below 30% on an idle machine, restart Ollama. When disk fills past 90%, fire a webhook. Nine built-in triggers ship active by default.

Actuation

Webhooks, MQTT, SSH commands

Webhooks with template variables, MQTT for Home Assistant, SSH commands on any node. Kill processes, restart services, reclaim GPUs — automatically or on demand.

OpenClaw Integration

Free openclaw-fleet plugin

Ask your AI agent "how's the fleet?" and get real-time answers. Proactive alerts in chat. github.com/leassh/openclaw-fleet

Blacklist & Process Mgmt

Kill unwanted processes

Blacklist games on shared machines. When a machine goes idle with Fortnite running, Leassh kills it and frees the GPU. Unknown processes? The LLM researches and suggests.

Trend Prediction

Predictive, not reactive

Rate-of-change analysis on disk, RAM, and temperature. "Disk losing 12 GB/h, critical in ~40 minutes." Not reactive — predictive.

For Families

Create your account

Sign up at leassh.com and get your unique family code. This code links all your family's computers to your dashboard.

Install the agent on each computer

Download the tiny Leassh agent, run the installer, and enter your family code. No configuration files, no terminal.

Open your dashboard

Screen time, limits, and content alerts are live immediately at app.leassh.com.

For Power Users

Install Leassh server

Docker or binary. docker run -d leassh/server and you're up. No cloud dependency.

Add machines to fleet.yaml

List your hosts, SSH credentials, and probe intervals. Or use the "Add Computer" UI in the dashboard.

Define rules and webhooks

Set up triggers, actuation, and limits in YAML or the UI. Fleet dashboard at :8383/fleet.

Prefer self-hosting? Run your own Leassh server (Docker or binary) and point agents at your server's address instead of our cloud. Full control, full privacy — your dashboard lives at your-server:8383/family.

Privacy you can actually verify

Most parental monitoring tools send your child's screen data to their cloud servers, where it sits in a database you can't audit. Leassh gives you a real choice.

Local Mode

Run a vision AI like Qwen on your own hardware. Screenshots are captured, analyzed, and summarized on your network. No data ever leaves your home. This is the gold standard of privacy.

Cloud Mode

Don't want to run infrastructure? Use our cloud. Screenshots are analyzed and immediately deleted — never stored in a database, never sold, never used for training. Just results, then gone.
